# Privacy Statement — RememberWindowsState

**Effective Date:** June 2025  
**Version:** 1.0  
**Application:** RememberWindowsState  
**Developer:** RememberWindowsState Contributors

---

## 1. Overview

RememberWindowsState ("the Application") is a lightweight Windows desktop utility designed to periodically snapshot your open application windows and restore them on demand. This Privacy Statement explains what data the Application accesses, how it is used, and what guarantees you have as a user.

**The short version:** RememberWindowsState does not collect, transmit, or share any personal data. Everything it reads stays on your local machine.

---

## 2. Data the Application Accesses

When running, RememberWindowsState reads the following information about your open application windows:

| Data Point | Purpose |
|---|---|
| **Window title** | Identify which application/document was open |
| **Executable file path** (`.exe`) | Relaunch the application during restore |
| **Window position and size** (`x`, `y`, `width`, `height`) | Restore windows to their original screen location |
| **Process name** | Filter out system/background processes; apply blacklist |

### What the Application does NOT access:

- ❌ Window **contents**, screen captures, or screenshots
- ❌ **Keystrokes**, clipboard data, or typed input
- ❌ **File contents** opened inside applications
- ❌ **Browser history**, URLs, or web activity
- ❌ **Passwords**, credentials, or authentication tokens
- ❌ **Personal files**, documents, images, or media
- ❌ **Microphone**, camera, or any hardware sensor
- ❌ **Network activity** or internet traffic

---

## 3. Data Storage

All data collected and generated by the Application is stored **locally on your device** in the following location:

```
%APPDATA%\RememberWindowsState\
├── config.json          ← Your settings (interval, blacklist, etc.)
├── windows_state.json   ← The latest window snapshot
└── logs\
    └── app.log          ← Application run logs (rotated at 200 KB, keeps up to 5 backups)
```

On a typical Windows installation this resolves to:

```
C:\Users\<YourName>\AppData\Roaming\RememberWindowsState\
```

**Storage guarantees:**

- Data and logs are stored in plain-text files that you can open, inspect, and delete at any time.
- The Application never encrypts data or logs in a way that prevents you from reading them.
- No database, cloud service, or remote server is involved.

---

## 4. Network Activity

RememberWindowsState makes **zero network connections** of any kind. It does not:

- Phone home or check for updates automatically
- Send telemetry, crash reports, or analytics
- Connect to any external API or web service
- Use web sockets, HTTP, or any other network protocol

You can verify this independently using a network monitor such as Windows Firewall, Wireshark, or Resource Monitor.

---

## 5. Windows Registry

When you enable **"Run at Windows Startup"** in Settings, the Application writes a single registry entry to:

```
HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run
```

Key name: `RememberWindowsState`  
Value: Path to the application executable with the `--startup` flag.

This entry is **removed automatically** when you disable startup from Settings, or you can delete it manually using `regedit` or Task Manager → Startup tab.

## 8. Your Rights and Control

Because the Application stores all data locally and collects no personal data, you retain full control at all times:

- **View your data and logs** — Open `windows_state.json`, `config.json`, and `logs\app.log` in any text editor.
- **Delete your data and logs** — Delete the `%APPDATA%\RememberWindowsState\` folder at any time.
- **Stop data collection** — Exit the Application from the System Tray.
- **Remove autostart** — Disable "Run at Windows Startup" in Settings, or remove the registry key manually.

To fully remove all traces of the Application:

1. Exit from System Tray → **Exit**
2. Uninstall via Windows **Settings → Apps** (if installed via installer)
3. Delete `%APPDATA%\RememberWindowsState\`
4. Remove the registry key (if autostart was enabled):  
   `HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run\RememberWindowsState`
